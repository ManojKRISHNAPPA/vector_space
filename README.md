# vector_space

A simple Streamlit app with:

- login and registration UI
- authenticated blog page
- AI DevOps blog content

## Run locally

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the app:

```bash
streamlit run app.py
```

3. Use the registration tab to create an account, then log in to view the AI DevOps blog.
